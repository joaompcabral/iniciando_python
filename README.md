Minha Jornada na Programação com Python 🚀
Bem-vindo ao meu repositório de estudos! Aqui estou documentando meu progresso saindo do zero até o nível profissional. Este espaço serve para guardar meus exercícios de lógica, refatorações e desafios superados.

 O que estou aprendendo:
Lógica de Programação: Entendimento de como estruturar o pensamento para resolver problemas.

Estruturas Condicionais: Uso de if, elif e else para tomadas de decisão inteligentes.

Laços de Repetição: Uso de for (repetições planejadas) e while (repetições baseadas em condições).

Vetores (Listas): Manipulação de coleções de dados, armazenamento dinâmico e sincronização entre listas diferentes.

Validação de Dados: Criação de "travas de segurança" com while para garantir a integridade das entradas.

Funções Estatísticas: Uso de métodos nativos como sum(), max() e len() para análise de dados.

Clean Code: Organizando o código para que seja fácil de ler, manter e profissional.

 Exercícios Atuais:
